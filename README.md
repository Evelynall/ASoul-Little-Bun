# 枝江小馒头

一个使用 PyQt6 制作的桌面宠物应用，类似 BongoCat，支持键盘动画和鼠标同步移动。

## 功能特性

- 透明无边框窗口，置顶显示
- 键盘按下时，键盘图片向下移动并回弹
- 鼠标移动时，鼠标图片在限定范围内同步移动
- 支持多角色切换（自动识别 img 目录下的文件夹）
- 可拖动窗口位置
- 右键菜单支持切换角色、设置和退出
- 设置界面支持自定义：
  - 窗口大小
  - 键盘图片位置和按下偏移量
  - 鼠标图片位置、移动范围和灵敏度
  - 设置自动保存到 config.json

## 安装依赖

```bash
pip install -r requirements.txt
```

## 目录结构

```
img/
├── 角色名1/
│   ├── bgImage.png        # 背景图片
│   ├── keyboardImage.png  # 键盘动画图片
│   └── mouseImage.png     # 鼠标动画图片
├── 角色名2/
│   ├── bgImage.png
│   ├── keyboardImage.png
│   └── mouseImage.png
...
```

## 运行

```bash
python main.py
```

## 使用说明

- 左键拖动窗口可以移动位置
- 右键点击窗口打开菜单
- 在菜单中选择"切换角色"可以切换不同的角色
- 在菜单中选择"设置"打开设置界面，可以调整：
  - 窗口大小（宽度和高度）
  - 键盘图片的 X/Y 偏移和按下时的偏移量
  - 鼠标图片的 X/Y 偏移、最大移动范围和移动灵敏度
  - 点击"保存"应用设置，点击"重置默认"恢复默认值
- 按下任意键盘按键，键盘图片会向下移动并回弹
- 移动鼠标，鼠标图片会在限定范围内同步移动
- 设置会自动保存到 config.json 文件中

## 注意事项

- 确保 img 目录下每个角色文件夹都包含五张图片：bgImage.png、keyboardImage.png、mouseImage.png、rightClickImage.png、leftClickImage.png
- 图片建议使用透明背景的 PNG 格式
- 默认窗口大小为 400x400，可以在设置界面中修改
- 设置保存在 config.json 文件中，首次运行会使用默认设置
